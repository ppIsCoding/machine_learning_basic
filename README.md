# 机器学习基础示例项目

这个项目包含了多个机器学习基础算法的实现，通过Python代码演示了各种核心概念和算法。

## 项目结构

- demo1.py - 简单线性回归
- demo2.py - 多项式回归
- demo3.py - 随机梯度下降法
- demo4.py - 多重回归
- demo5.py - 感知机（分类问题）
- demo6.py - 逻辑回归（分类）
- demo7.py - 处理线性不可分情况
- demo8.py - 正则化防止过拟合

## 依赖库

- numpy
- matplotlib

## 项目介绍

### demo1.py - 简单线性回归
使用梯度下降法实现简单的一次函数线性回归，对数据进行标准化预处理，通过迭代优化参数来最小化误差。

### demo2.py - 多项式回归
扩展到多项式函数，实现二次函数拟合，并展示了误差变化图，演示了如何使用梯度下降法优化多项式回归模型。

### demo3.py - 随机梯度下降法
与批量梯度下降不同，使用随机梯度下降法更新参数，每次迭代随机选择样本进行参数更新，提高训练效率。

### demo4.py - 多重回归
实现多重回归算法，处理多个特征变量的回归问题，与多项式回归不同，它处理的是多个不同的输入变量。

### demo5.py - 感知机（分类问题）
实现感知机算法解决二分类问题，使用线性分类器，通过调整权重向量来找到决策边界。

### demo6.py - 逻辑回归（分类）
使用逻辑回归解决分类问题，通过sigmoid函数将线性回归输出转换为概率，适用于二分类任务。

### demo7.py - 处理线性不可分情况
针对线性不可分的数据，通过增加特征维度（如二次项）来处理非线性可分问题，并使用随机梯度下降优化。

### demo8.py - 正则化防止过拟合
演示了多项式回归中的过拟合问题，并通过L2正则化（Ridge回归）来防止过拟合，提高模型泛化能力。

## 使用方法

1. 安装依赖库：
```bash
pip install numpy matplotlib
```

2. 确保有必要的数据文件（如click.csv, images1.csv, images2.csv, data3.csv等）

3. 运行示例：
```bash
python demo1.py
python demo2.py
# ... 依此类推
```

## 数据文件

项目中的示例使用了以下数据文件：
- click.csv - 用于回归分析的点击数据
- images1.csv - 用于二分类问题的图像数据
- images2.csv - 用于逻辑回归的图像数据
- data3.csv - 用于处理线性不可分情况的数据

## 学习目标

通过这些示例，你可以学习到：
- 线性回归和多项式回归的实现
- 梯度下降算法的原理和应用
- 分类算法（感知机、逻辑回归）
- 过拟合问题及正则化解决方案
- 数据预处理技术（标准化）

## 算法原理

每个示例都包含了详细的注释，解释了算法的数学原理和实现步骤，适合机器学习初学者理解和学习。
